## Interação com o Cliente

### Canais de Comunicação

<ul class="softex-list">
  <li>**Revisões Quinzenais**: Validação incremental ao final de cada sprint</li>
  <li>**Canal Slack**: Comunicação contínua 24/7</li>
  <li>**Google Meet**: Reuniões formais para apresentações</li>
  <li>**GitHub**: Acompanhamento técnico transparente</li>
</ul>

### Fluxo de Validação

<ol class="softex-flow">
  <li>🚀 **Desenvolvimento** da funcionalidade</li>
  <li>📹 **Apresentação** no Google Meet</li>
  <li>💬 **Feedback** do cliente via Slack</li>
  <li>🔧 **Ajustes** baseados no retorno</li>
  <li>✅ **Validação** final e aprovação</li>
</ol>
